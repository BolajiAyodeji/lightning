    View Source Lightning.Runtime.Handler behaviour (Lightning v0.4.3)

    A strategy for executing things via ChildProcess.

    This module handles the dirty bits, setting up processes and coordinating results (and logs) as they arrive.

    Since it is a macro, see Lightning.Pipeline.Runner.Handler for a usage example.
